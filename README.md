# monogame-funny-game

A simple tower defense game built with MonoGame (DesktopGL).

## Concept

Enemies follow a path across the map. Place towers along the path to defeat them before they reach the end. Survive all waves to win.

## Requirements

- .NET 9 SDK
- MonoGame 3.8

## How to Run

```bash
dotnet restore
dotnet run
```
